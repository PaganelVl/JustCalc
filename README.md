Простой калькулятор на <a href="">PySide6</a>

<h1>Функции:</h1>

<ul>
<li>Сложение</li>
<li>Вычитание</li>
<li>Умножение</li>
<li>Деление</li>
<li>Вычисление процента</li>
<li>Возведение в степень</li>
<li>Нахождение модуля (|x|)</li>
<li>Деление единицы</li>
</ul>

<h1>Требования</h1>
<code>pip install -r requirements.txt</code>
